awsutil bastions - List and configure bastion hosts

USAGE:
    awsutil bastions [--profile <aws cli profile>]
    awsutil bastions [-p <aws cli profile>]

DESCRIPTION:
    Lists all configured bastions for the specified profile. Provides
    an interactive interface to configure new bastions by querying AWS
    for RDS databases and EC2 bastion instances.

OPTIONS:
    --profile, -p    AWS CLI profile to use

INTERACTIVE MODE:
    After listing existing bastions, you can choose to configure a new
    bastion. The tool will:
    1. Query and display available RDS databases
    2. Query and display available bastion EC2 instances
    3. Allow you to select a database and bastion instance
    4. Auto-generate a bastion name or prompt for one
    5. Auto-find an available local port
    6. Save the configuration
